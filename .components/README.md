# .components/ — 21st.dev snippets & custom components для CRES-CA

Сюда кидаем найденные на 21st.dev или собственные component-заготовки **ДО** интеграции в `app/src/components/`. После того как компонент попал в реальный код и прижился — можно удалить отсюда или промотнуть в `D:/toolbox/components/<category>/`.

Категории: `layout/ forms/ cards/ modals/ data-display/ marketing/ motion/` (см. `D:/toolbox/components/README.md` для правил).
